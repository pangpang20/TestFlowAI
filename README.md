# TestFlowAI

基于 AI 智能体的自动化测试工具

## 功能特性

- 🤖 智能体驱动的测试流程
- 📋 自动化测试用例生成
- 🔗 支持多种测试框架集成
- 📊 测试报告自动生成

## 快速开始

```bash
# 克隆项目
git clone git@github.com:pangpang20/TestFlowAI.git

# 进入目录
cd TestFlowAI

# 安装依赖
npm install

# 运行测试
npm test
```

## 技术栈

- Node.js / Java
- Spring Boot
- SQLite / MySQL / PostgreSQL
